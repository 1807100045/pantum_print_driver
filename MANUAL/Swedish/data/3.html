3. Installation och avinstallation av drivrutin

3.1. Hur du installerar Windows skrivardrivrutin

Windows drivrutin installationsprogram ger två installationsmetoder: ett-klicks sinstallation och manuell installation.Vi rekommenderar ett-klicksinstallation vilken kan hjälpa till att installera drivrutinen automatiskt på ett snabbare och bekvämare sätt.Om du får problem med att använda ett-klicksinstallationen var god prova manuell installation.

Obs!

• Det finns skillnader mellan installationsgränssnitt på grund av typ och funktioner; skissen är bara som referens.

3.1.1. Ett-klicks installation

Ett-klicks installation ger tre metoder för installation: anslut skrivaren via USB kabel, anslut skrivaren via trådbundet nätverk och anslut skrivaren via trådlöst nätverk. Du kan välja det som du är van vid.

3.1.1.1. Metod I: Anslut skrivaren via USB-kabel

1. Anslut skrivaren till datorn via USB-kabel sätt på skrivaren och din dator.

2. Placera medföljande installations-CD i din dators CD-ROM:

• Windows XP system: kör installationen automatiskt.

• Windows 7 / Vista / Server 2008-systemet: Pop up "Autoplay" -gränssnittet, klicka på "Autorun.exe" för att köra installationsproceduren.

• Windows 8 system: När fönstret "DVD RW driver" dyker upp i översta högra hörnet på dator-skrivbordet klicka var som helst i pop-up fönstret, klicka sedan på "Autorun.exe" för att köra installationen.

• Windows 10 system: När fönstret "DVD RW driver" fönster dyker upp i översta högra hörnet på dator-skrivbordet klicka var som helst i pop-up fönstret, klicka sedan på "Autorun.exe" för att köra installationen.

Obs!

• Några datorer kanske inte kan köra auto play CD beroende på systemkonfiguration. Var god dubbelklicka på Dator-ikonen och hitta "DVD RW driver", dubbelklicka sedan på "DVD RW driver" för att köra installationen.

3. Klicka på knappen "Jag godkänner" längst ner i högra hörnet efter att ha läst licens-överenskommelsen och ange gränssnittet för drivrutinsinstallation.

4. Klicka på knappen "En-klick-installationen" i Metod I och kör installationsprocessen. Processen kan ta lite tid, och hur lång tid är beroende av konfigurationen på din dator, var god och vänta tålmodigt.

5. Efter installationen, klicka på knappen "Skriv ut testsida". Om din skrivare lyckas skriva ut testsidan, betyder det att du har installerat drivrutinen korrekt.

3.1.1.2. Metod II: Anslut skrivare via trådlöst nätverk network

1. Slå på strömmen till skrivarens dator.

2. Anslut datorn till det trådbundna nätverket.

3. Anslut nätverkskabeln till skrivarens internetport och se till att skrivaren ansluts till datorns nätverk (Se Kapitel 4 för hur du konfigurerar internet).

4. Placera medföljande installations-CD i din dators CD-ROM:

• Windows XP system: kör installationen automatiskt.

• Windows 7 / Vista / Server 2008-systemet: Pop up "Autoplay" -gränssnittet, klicka på "Autorun.exe" för att köra installationsproceduren.

• Windows 8 system: När fönstret "DVD RW driver" dyker upp i översta högra hörnet på dator-skrivbordet klicka var som helst i pop-up fönstret, klicka sedan på "Autorun.exe" för att köra installationen.

• Windows 10 system: När fönstret "DVD RW driver" fönster dyker upp i översta högra hörnet på dator-skrivbordet klicka var som helst i pop-up fönstret, klicka sedan på "Autorun.exe" för att köra installationen.

Obs!

• Några datorer kanske inte kan köra auto play CD beroende på systemkonfiguration. Var god dubbelklicka på Dator-ikonen och hitta "DVD RW driver", dubbelklicka sedan på "DVD RW driver" för att köra installationen.

5. Klicka på knappen "Jag godkänner" längst ner i högra hörnet efter att ha läst licens-överenskommelsen och ange gränssnittet för drivrutinsinstallation.

6. Var god klicka på knappen "En-klick-installationen" i Metod II och installera drivrutinen.

Obs!

• Du kan ställa musen på knappen "En-klick-installationen" för att kolla popup information och bekräfta om den aktuella skrivaren är den du behöver installera genom IP address innan du klickar på knappen.

• Du kan kontrollera skrivarens IP adress genom sidan "Nätverkskonfiguration" (Vänligen se Kapitel 8.1.1.3 om hur man skriver ut).

• Klicka på knappen "En-klick-installationen" för att starta installationsprocessen om din programvara endast har hittat en skrivare. Processen kan ta lite tid, och och beror på konfiguration på din dator . Vänligen vänta tålmodigt.

• Om din programvara har sökt flera skrivare klickar du på knappen "En-klick-installationen", och skrivarlistans gränssnitt kommer att dyka upp. Välj den skrivare du behöver enligt din IP-adress eller värd-ID för skrivaren.

Obs!

• Om skrivaren du behöver inte visas i listan, vänligen klicka på "Refresh" knappen för att söka igen. Om den inte hittas, vänligen kontrollera om skrivare och dator är är i samma nätverk (Vänligen se Kapitel 5.3 om hur man kontrollerar).

• När skrivarens IP address ändras under skrivardrivrutinen som är installerad med host ID, kan du också direkt skriva ut utan att installera om skrivaren. Om du behöver installera skrivarens drivrutin med värd ID, vänligen kontrollera alternativet "Installera skrivaren i listan med värd ID".

7. Efter installationen, klicka på knappen "Skriv ut testsida". Om din skrivare lyckas skriva ut testsidan, betyder det att du har installerat drivrutinen korrekt.

3.2.1. Installation av drivrutin

Mac 10.11 används som ett exempel för följande operationer, det kan finnas skillnader i din datorskärminformation på grund av olika operativsystem.

1. Slå på strömmen till skrivarens dator.

2. Sätt i den medföljande installations-CD:n i CD-enheten på datorn, dubbelklicka på "Pantum XXX-serien" installationspaket. ("XXX" i Pantum XXX-serien står för produkttyp).

3. Klicka på ” Fortsätt”.

4. Läs licensavtalet och klicka sedan på "Fortsätt".

5. Klicka på ”Godkänner" i popup-fönstret och acceptera Licensavtalet.

6. Klicka på ”Installation”.

7. Ange datorns lösenord och klicka på ”Installera programvara”.

8. Klicka på ”Fortsätt installationen" i popup-fönstret.

9. Systemet slutför automatiskt drivrutininstallationen.

• Om USB-kabel används för att ansluta datorn och Wi-Fi-dator, kommer följande konfigurationsfönster för trådlöst nätverk att dyka upp under drivrutininstallationen.

Obs!

• Klicka på "Ja", det trådlösa nätverket på skrivaren kan konfigureras av konfigurationsverktygen för trådlösa nätverk (Se Kapitel 5.1.1om hur du konfigurerar). Efter avslutad konfiguration tar du bort USB-kabel och lägger till skrivaren med nätverk som är anslutet i popup-gränssnittet "Lägg till skrivaren". (Se Kapitel 3.3 om hur du lägger till skrivare).

• Klicka på "Nej", skrivaren med USB-anslutning kan läggas till i gränssnittet "Lägg till skrivare" (Se Kapitel 3.3 om hur du lägger till skrivare).

Om datorn och icke-Wi-Fi-dator är anslutna via USB-anslutningskabel kommer gränssnittet för "Lägg till skrivare" att dyka upp under drivrutininstallationen och skrivaren kan läggas till på denna plats, se Kapitel 3.3 om hur du lägger till skrivare.

3.3. Lägg till skrivare i Mac-system

3.3.1. Lägg till skrivaren via USB

1. Anslut skrivaren och datorn med USB-anslutningskabel och slå på strömmen.

2. Ange datorns "Systeminställningar" för "Skrivare och skanner".

3. Klicka på knappen och välj "Lägg till skrivare och skanner" .

4. Välj skrivare och välj sedan motsvarande skrivartyp i popup-menyn "Använd".

5. Klicka på "Lägg till".

3.3.2. Lägg till skrivaren via trådbunden anslutning

1. Slå på strömmen till skrivarens dator.

2. Anslut datorn till det trådbundna nätverket.

3. Anslut nätverkskabeln till skrivarens internetport och se till att skrivaren ansluts till datorns nätverk (Se Kapitel 4 för hur du konfigurerar internet).

4. Ange datorns "Systeminställningar" för "Skrivare och skanner".

5. Klicka på knappen och välj "Lägg till skrivare och skanner" .

6. Välj skrivare och välj sedan motsvarande skrivartyp i popup-menyn "Använd".

7. Klicka på "Lägg till".

3.3.3. Lägg till skrivaren via trådlös anslutning

1. Slå på strömmen till skrivarens dator.

2. Se till att den trådlösa nätverksanslutningen är etablerad mellan datorn och skrivaren.

Obs!

• Om det är infrastrukturläge, anslut den trådlösa nätverksskrivaren till åtkomstpunkten (trådlös router); Se Kapitel 5.1 om hur du ansluter.

• Om det är Wi-Fi hot-point-läge, ansluter du den trådlösa klienten direkt till den trådlösa nätverksskrivaren. Se Kapitel 5.2 för hur du ansluter.

3. Ange datorns "Systeminställningar" för "Skrivare och skanner".

4. Klicka på knappen och välj "Lägg till skrivare och skanner" .

5. Välj skrivare och välj sedan motsvarande skrivartyp i popup-menyn "Använd".

6. Klicka på "Lägg till".
