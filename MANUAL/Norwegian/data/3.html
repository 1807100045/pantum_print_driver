3. Installasjon og lossing a driveren

3.1. Hvordan installere Windows printer driver

Windows driver installasjonsprogramvare tilbyr to installasjonsmetoder: ett-klikks installasjon og manuell installasjon.Vi anbefaler ett-klikks installasjon, som kan hjelpe å installere driveren automatisk raskere og mer beleilig.Hvis du har problemer med ett-klikks installasjon, vennligst prøv manuell installasjon.

Merk:

• Det er forskjeller mellom installasjonsgrensesnittene på grunn a typene og funksjonene; Skisse kartet er bare for referanse.

3.1.1. Ett-klikks Installasjon

Ett-klikks Installasjon tilbyr tre metoder for installasjon: koble til printeren via USB kabel, koble til printeren via kablet nettverk, og koble til printeren via trådløst nettverk. Du velger selv den som passer deg best.

3.1.1.1. Metode I: Koble til printeren via USB kabel

1. Koble printeren til Datamaskinen via USB kabel, slå på printeren og Datamaskinen.

2. Legg medfulgt installasjons CD i datamaskinens CD-ROM:

•Windows XP system: kjør installasjon automatisk.

• Windows 7/Vista/Server 2008 system: Pop up “Autokjør” grensesnitt, klikk “Autorun.exe” for å starte installasjon.

• Windows 8 system: Når “DVD RW driver” vinduet popper opp fra høyre hjørne av datamaskinens skrivebord, klikk hvor som helst i pop-up vinduet, klikk så på “Autorun.exe” for å starte installasjonen.

• Windows 10 system: Når “DVD RW driver” vinduet popper opp fra høyre hjørne av datamaskinens skrivebord, klikk hvor som helst i pop-up vinduet, klikk så på “Autorun.exe” for å starte installasjonen.

Merk:

• Enkelte Datamaskinenser kan ikke automatisk spille av CD-en grunnet system konfigurasjonen. Vennligst dobbeltklikk på "Computer" ikonet, finn "DVD RW driver", og dobbeltklikk "DVD RW driver" for å kjøre installasjonen.

3. Klikk på "Jeg godtar" -knappen nederst i høyre hjørne etter å ha lest lisensavtalen og gå inn i installasjonsgrensesnittet til driveren.

4. Klikk på “Ett-klikk installasjon" -knappen i Metode I og gå inn i installasjonsprosessen. Prosessen kan trenge litt tid, og varigheten er relatert til Datamaskin-konfigurasjonen, vennligst vent.

5. Etter installasjonen klikker du på "Skriv ut testside" -knappen. Hvis skriveren skriver ut testsiden vellykket, betyr det at du har installert driveren riktig.

3.1.1.2. Metode II: Koble printeren via kablet nettverk

1. Slå på strømmen til skriveren datamaskinen.

2. Koble datamaskinen til det kablete nettverket.

3. Koble nettverkskabelen til internettporten til skriveren, og kontroller at skriveren kobles til datamaskinens nettver (se kapittel 4 for hvordan du kan konfigurere internett).

4. Legg medfulgt installasjons CD i datamaskinens CD-ROM:

• Windows XP system: kjør installasjon automatisk.

• Windows 7/Vista/Server 2008 system: Pop up “Autokjør” grensesnitt, klikk “Autorun.exe” for å starte installasjon.

• Windows 8 system: Når “DVD RW driver” vinduet popper opp fra høyre hjørne av datamaskinens skrivebord, klikk hvor som helst i pop-up vinduet, klikk så på “Autorun.exe” for å starte installasjonen.

• Windows 10 system: Når “DVD RW driver” vinduet popper opp fra høyre hjørne av datamaskinens skrivebord, klikk hvor som helst i pop-up vinduet, klikk så på “Autorun.exe” for å starte installasjonen.

Merk:

• Enkelte Datamaskinenser kan ikke automatisk spille av CD-en grunnet system konfigurasjonen. Vennligst dobbeltklikk på "Computer" ikonet, finn "DVD RW driver", og dobbeltklikk "DVD RW driver" for å kjøre installasjonen.

5. Klikk på "Jeg godtar" -knappen nederst i høyre hjørne etter å ha lest lisensavtalen og gå inn i installasjonsgrensesnittet til driveren.

6. Vennligst klikk på “Ett-klikk installasjon” -knappen i Metode II og installer driver programvare.

Merk:

• Du kan holde musepekeren over “Ett-klikk installasjon” -knappen, før du klikker på den, for å sjekke popup-informasjonen for å bekrefte om den nåværende skriveren er den du trenger å installere gjennom IP-adressen.

• Du kan sjekke printerens IP-adresse gjennom “Nettverk konfigurasjon side” (Vennligst se kapittel 8.1.1.3 for hvordan man printer).

• Klikk på “Ett-klikk installasjon" -knappen for å gå inn i installasjonsprosessen hvis programvaren bare har søkt en skriver. Prosessen kan trenge litt tid, og varigheten er relatert til Datamaskinens-konfigurasjonen. Vennligst vent.

• Hvis programvaren har søkt flere skrivere, klikker du på “Ett-klikk installasjon" -knappen, og grensesnittet til skriverlisten kommer opp. Vennligst velg skriveren du trenger i henhold til din IP-adresse eller host-ID for skriveren.

Merk:

• Hvis printeren din ikke er vist i listen, vennligst klikk “Oppdater” -knappen for å søke på nytt. Hvis den fremdeles ikke kommer opp i listen, vennligst sjekk om printeren og Datamaskinen er på samme nettverk(Vennligst sjekk kapittel 5.3 for hvordan man sjekker nettverket).

• Når skriverens IP-adresse endres under skriverdriveren installert med verts-ID, kan du også skrive ut direkte uten å installere driveren på nytt. Hvis du trenger å installere skriverdriveren med verts-ID, vennligst sjekk alternativet "Installer skriveren i listen med verts ID".

7. Etter installasjonen klikker du på "Skriv ut testside" -knappen. Hvis skriveren skriver ut testsiden vellykket, betyr det at du har installert driveren riktig.

3.1.2.3.1. Installasjon av infrastrukturmodus

1. Koble datamaskinen til tilgangspunktet (trådløs ruteren).

2. Koble skriveren til Datamaskinen via USB kabel, slå så på printeren.

3. Legg medfulgt installasjons CD i datamaskinens CD-ROM:

• Windows XP system: kjør installasjon automatisk.

• Windows 7/Vista/Server 2008 system: Pop up “Autokjør” grensesnitt, klikk “Autorun.exe” for å starte installasjon.

• Windows 8 system: Når “DVD RW driver” vinduet popper opp fra høyre hjørne av datamaskinens skrivebord, klikk hvor som helst i pop-up vinduet, klikk så på “Autorun.exe” for å starte installasjonen.

• Windows 10 system: Når “DVD RW driver” vinduet popper opp fra høyre hjørne av datamaskinens skrivebord, klikk hvor som helst i pop-up vinduet, klikk så på “Autorun.exe” for å starte installasjonen.

Merk:

• Enkelte Datamaskinenser kan ikke automatisk spille av CD-en grunnet system konfigurasjonen. Vennligst dobbeltklikk på "Computer" ikonet, finn "DVD RW driver", og dobbeltklikk "DVD RW driver" for å kjøre installasjonen.

4. Klikk på "Jeg godtar" -knappen nederst i høyre hjørne etter å ha lest lisensavtalen og gå inn i installasjonsgrensesnittet til driveren.

5. Klikk på "manuell installasjon" nederst på driverens installasjonsgrensesnitt, og gå inn i grensesnitt for manuell tilkobling.

6. Vennligst velg installasjons språk og skrivermodell.

7.Velg "Skrivere tilkoblet et nytt nettverk" og klikk på "Installer".

8. Velg "Ja, jeg skal konfigurere det trådløse nettverket til skriveren'' i det trådløse konfigurasjonsinduet, betjene som instruksjonene og konfigurere skriveren til det trådløse nettverket (se kapittel 5.1.1 for hvordan du kan konfigurere).

9.Installasjonsprogramvaren vil søke etter skriveren automatisk etter at trådløst nettverk er konfigurert. Søkeprosessen kan trenge litt tid.

10. Velg skriveren som trenger å koble til og klikk på "Neste''.

Merk:

• Hvis det ikke er noen skriver du trenger å koble til i den undersøkte skriverlisten, vennligst, klikk på "Oppdater for å Søke'' igjen.

• Når skriverens IP-adresse endres under skriverdriveren installert med verts-ID, kan du også skrive ut direkte uten å installere driveren på nytt. Hvis du trenger å installere skriverdriveren med verts-ID, vennligst sjekk alternativet "Installer skriveren i listen med verts ID".

• Hvis IP-adressen og vertsnavnet legges til manuelt, skal du kjenne IP-adressen eller vertsnavnet til skriveren som skal kobles til. Hvis det ikke er klart, skriv ut "Internettkonfigurasjonsside'' og "Vertsnan'' (Vennligst se Kapittel 8.1.1.3 for hvordan man printer).

11. Programvaren installerer driveren automatisk, og installasjonsprosessen kan trenge litt tid. Varigheten er relatert til Datamaskin-konfigurasjonen, vennligst vent.

12. Etter installasjonen klikker du på "Skriv ut testside" -knappen. Hvis skriveren skriver ut testsiden vellykket, betyr det at du har installert driveren riktig.

3.1.3. Metode for avinstallasjon av driver

Windows 7 brukes som et eksempel på følgende operasjoner, det kan være forskjeller i datamaskjerm informasjonen din på grunn a det forskjellige operativsystemet.

1. Klikk på "Start-menyen'' på datamaskinen, og klikk på deretter på "Alle programmer''.

2. Klikk på "Pantum'', og klikk på deretter "Pantum XXX-serien''. ("XXX" i Pantum XXX-serien står for profukttype).

3. Klikk "Laste opp", og fjern skriveren i henhold til instruksjonen laste opp induet.

4. Starte datamaskinen på nytt etter at du har fullført laste opp.

3.2.Mac system driver installasjon

• Driver installasjon under Mac-systemet er delt inn i driverinstallasjonen og legger til skriver.

• Når du installerer driveren ved hjelp a kablet eller trådløst nettverk, skal du kjenne Bonjour-navnet til skriveren, vennligst skriv it og sjekk "Internettkonfigurasjonsside'' (Vennligst se Kapittel 8.1.1.3 for hvordan man printer).

3.2.1. Driver installasjon

Mac 10.11 brukes som et eksempel på følgende operasjoner, det kan være forskjeller på skjermen din på grunn av det forskjellige operativsystemet.

1. Slå på strømmen til skriveren datamaskinen.

2. Sett inn den vedlagte installasjons-Cden i CD-driveren på datamaskinen, og dobbeltrikk på "Pantum XXX serien'' installasjonspakken. ("XXX" i Pantum XXX-serien står for produkttype.

3. Klikk på "fortsett''.

4. Les lisensatalen, og klikk på deretter "Fortsett''.

5. Klikk på "Enig'' i pop-opp vinduet, og godta lisensatalen.

6. Klikk på "Installasjon''.

7. Put inn datamaskinens passord og klikk på "Installer programvare''.

8. Klikk på "Fortsett installasjon'' i pop-opp vinduet.

9. Systemet vil automatisk fullføre driverinstallasjonen.

Hvis USB-tilkoblingslinjen brukes til å koble til datamaskinen og Wi-Fi datatypen, vises følgende konfigurasjonsindu for trådløst nettverk under prosesses med driverinstallasjon.

Merk:

• Klikk på "ja'', det trådløse nettverket til skriveren kan konfigureres av konfigurasjonsverktøyene til det trådløse nettverket (se kapittel 5.1.1for hvordan du kan konfigurere). Etter at konfigurasjonen er fullført, dukk opp USB-tilkoblingslinjen og legg skriveren med nettver koblet til i dukk-opp grensesnittet "Legg til skriveren''. (vennligst se kapittel 3.3 for hvordan du kan legge til skriveren).

• Klikk på "Nei'', skriveren med USB koblet kan legges til i grensesnittet til "Legg til skriver'' (vennligst se kapittel 3.3 for hvordan du kan legge til skriveren).

Hvis datamaskinen og ikke-Wi-Fi datatypen er koblet til via USB-tilkoblingslinje, vil grensesnittet til "Legg til skriver'' dukke opp under prosessen til driverinstallasjonen, og skriveren kan legges til på dette stedet. Vennligst se kapittel 3.3 for hvordan du kan legge til skriveren.

3.3.Legg til skriver i Mac-system

3.3.1. Legg til skriveren via USB

1. Koble skriveren og datamaskinen med USB-tilkoblignslinjen og slå på strømmen.

2. Skriv inn "Systemvalginnstilling'' til datamaskinen - "Skriver og Skanner''.

3. Klikk på knappen og velg "Legg til skriver og skanner'' .

4. Velg skriveren, og velg deretter den tilhørende skrivertypen i dukk opp menyen "Bruk''.

5. Klikk på "legg til''.

3.3.2. Legg til skriveren via kablet tilkobling

1. Slå på strømmen til skriveren datamaskinen.

2. Koble datamaskinen til det kablete nettverket.

3. Koble nettverkskabelen til internettporten til skriveren, og kontroller at skriveren kobles til datamaskinens nettver (se kapittel 4 for hvordan du kan konfigurere internett).

4. Skriv inn "Systemvalginnstilling'' til datamaskinen - "Skriver og Skanner''.

5. Klikk på knappen og velg "Legg til skriver og skanner'' .

6. Velg skriveren, og velg deretter den tilhørende skrivertypen i dukk opp menyen "Bruk''.

7. Klikk på "legg til''.

3.3.3. Legg til skriveren via trådløs tilkobling

1. Slå på strømmen til skriveren datamaskinen.

2. Kontroller at den trådløse nettverksforbindelse er etablert mellom datamaskinen og skriveren.

Merk:

• Hvis det er infrastrukturmodus, koble du den trådløse nettverksskriveren til tilgangspunktet (trådløs router); Vennligst se kapittel 5.1 for hvordan du kan koble til.

• His det er Wi-Fi-hotspotmodus, koble den trådløse klienten direkte til den trådløse nettverksskriveren. Vennligst se kapittel 5.2 for hvordan du kan koble til.

3. Skriv inn "Systemvalginnstilling'' til datamaskinen - "Skriver og Skanner''.

4. Klikk på knappen og velg "Legg til skriver og skanner'' .

5. Velg skriveren, og velg deretter den tilhørende skrivertypen i dukk opp menyen "Bruk''.

6. Klikk på "legg til''.
