3. 驅動安裝與卸載

3.1. Windows系統的驅動安裝

Windows驅動安裝軟體提供一鍵安裝和手動安裝兩種安推薦您使用一鍵安裝方式，它可以説明您更快速、更便捷的自動完成驅動安裝。當您使用一鍵安裝方式遇到困難時，您可以嘗試手動安裝。

注：

• 驅動安裝介面因型號功能不同存在差異，示意圖僅供參考。

3.1.1. 一鍵安裝

一鍵安裝為您提供USB線連接印表機、有線網路連接印表機、無線網路連接印表機三種安裝方式，您可以根據您印表機支援的連接方式選擇您習慣使用的安裝方式。

3.1.1.1. 方式一：USB線連接印表機

1. 使用USB線連接印表機和電腦，並開啟印表機和電腦電源。

2. 在電腦的光碟機中插入隨附的安裝光碟：

• Windows XP系統：自動運行安裝程式。

• Windows 7/Vista/Server 2008系統：彈出“自動播放”介面，點擊“Autorun.exe”，運行安裝程式。

• Windows 8系統：電腦桌面右上角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。

•Windows 10系統：電腦桌面右下角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。s

注：

• 部分電腦可能因系統組態等原因，插入安裝光碟後不會自動播放光碟，請按兩下“電腦”，找到“DVD RW 驅動器”，按兩下“DVD RW 驅動器”，運行安裝程式。

3. 閱讀授權合約，然後點擊介面右下角的“我同意”按鈕，進入驅動安裝介面。

4. 點擊方式一下方的“一鍵安裝”按鈕，進入安裝過程，安裝過程可能需要一定時間，時間長短與您電腦配置有關，請您耐心等待。

5. 安裝完成，點擊“列印測試頁”，如果您的印表機能列印測試頁，說明您已經正確安裝印表機驅動。

3.1.1.2.方式二：有線網路連接印表機

1. 打開印表機和電腦的電源。

2. 將電腦連接到有線網路。

3. 將網線連接到印表機的網路介面，確保印表機連接到電腦所在的網路（如何配置網路，請參閱第4章）。

4. 在電腦的光碟機中插入隨附的安裝光碟：

• Windows XP系統：自動運行安裝程式。

• Windows 7/Vista/Server 2008系統：彈出“自動播放”介面，點擊“Autorun.exe”，運行安裝程式。

• Windows 8系統：電腦桌面右上角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。

• Windows 10系統：電腦桌面右下角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。

注：

• 部分電腦可能因系統組態等原因，插入安裝光碟後不會自動播放光碟，請按兩下“電腦”，找到“DVD RW 驅動器”，按兩下“DVD RW 驅動器”，運行安裝程式。

5. 閱讀授權合約，然後點擊介面右下角的“我同意”按鈕，進入驅動安裝介面。

6. 點擊方式二下方的“一鍵安裝”按鈕進行驅動軟體安裝。

注：

• 在點擊“一鍵安裝”前，您可以先將滑鼠游標放置在“一鍵安裝”按鈕上，查看彈出懸浮窗的提示資訊，通過IP位址確認當前印表機是否是您想要安裝的印表機。

• 您可以通過列印“網路配置頁”查看印表機的IP位址（如何列印，請參閱第8.1.1.3章）。

• 如果此時您的安裝軟體只搜索到一台印表機，點擊“一鍵安裝”按鈕，進入安裝過程，安裝過程可能需要一定時間，時間長短與您電腦配置有關，請您耐心等待。

• 如果此時您的安裝軟體搜索到多台印表機，點擊“一鍵安裝”按鈕，會彈出印表機清單介面，請您根據您印表機的有線網路的IP位址或主機名稱選擇您需要安裝的印表機。

注：

• 若印表機清單介面未顯示您需要安裝的印表機，請點擊“刷新”按鈕重新搜索。若仍搜索不到，請檢查印表機與電腦是否在同一網路（如何檢查，請參閱第5.3章）。

• 主機名稱安裝印表機驅動在印表機IP位址發生變化時，您還可以直接列印，不需要重新安裝驅動。若需要以主機名稱方式安裝印表機驅動，請勾選 “以主機名稱安裝列表中的印表機”。

7. 安裝完成，點擊“列印測試頁”，如果您的印表機能列印測試頁，說明您已經正確安裝印表機驅動。

3.1.1.3. 方式三：無線網路連接印表機

在安裝印表機驅動軟體時，請先檢查印表機是否已連接到無線網路，若未連接，請參閱第3.1.1.3.1章進行安裝；若已連接，請參閱第3.1.1.3.2章進行安裝。

注：

• 如何判斷印表機是否已連接網路，您可以通過列印“網路配置頁”確認印表機的網路連接狀態（如何列印，請參閱第8.1.1.3章）。

3.1.1.3.1. 印表機未連接到無線網路

3.1.1.3.1.1. 通過USB連接線配置無線網路並安裝驅動

因某些電腦不具備無線網路功能，故本節分為“電腦通過Wi-Fi連接至路由器”和“電腦通過網線連接至路由器”兩種場景來介紹安裝過程。

I．電腦通過Wi-Fi連接至路由器

1. 電腦已通過Wi-Fi連接至路由器。

2. 使用USB連接線將印表機連接至電腦，並開啟印表機電源。

3. 在電腦的光碟機中插入隨附的安裝光碟：

• Windows XP系統：自動運行安裝程式。

• Windows 7/Vista/Server 2008系統：彈出“自動播放”介面，點擊“Autorun.exe”，運行安裝程式。

•Windows 8系統：電腦桌面右上角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。

• Windows 10系統：電腦桌面右下角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。

注：

• 部分電腦可能因系統組態等原因，插入安裝光碟後不會自動播放光碟，請按兩下“電腦”，找到“DVD RW 驅動器”，按兩下“DVD RW 驅動器”，運行安裝程式。

4. 閱讀授權合約，然後點擊介面右下角的“我同意”按鈕，進入驅動安裝介面。

5. 點擊方式三下方的“一鍵安裝”按鈕進行驅動軟體安裝。

注：

• 在點擊“一鍵安裝”前，您可以先將滑鼠游標放置在“一鍵安裝”按鈕上，查看彈出懸浮窗的提示資訊，通過IP位址確認當前印表機是否是您想要安裝的印表機。

• 您可以通過列印“網路配置頁”查看印表機的IP地址（如何列印，請參閱第8.1.1.3章）。

• 如果此時您的安裝軟體只搜索到一台印表機，點擊“一鍵安裝”按鈕，進入安裝過程自動完成網路配置及驅動安裝。

• 如果此時您的安裝軟體搜索到多台印表機，點擊“一鍵安裝”按鈕，會彈出印表機清單介面，請您選擇“通過USB給印表機配置Wi-Fi並安裝”並勾選USB印表機，點擊“確定”按鈕，進入安裝過程自動完成網路配置及驅動安裝。

注：

• 主機名稱安裝印表機驅動在印表機IP位址發生變化時，您還可以直接列印，不需要重新安裝驅動。若需要以主機名稱方式安裝印表機驅動，請勾選 “以主機名稱安裝列表中的印表機”。

6. 安裝完成，點擊“列印測試頁”，如果您的印表機能列印測試頁，說明您已經正確安裝印表機驅動。

II．電腦通過網線連接至路由器

1. 將電腦通過網線連接至路由器。

2. 使用USB連接線將印表機連接至電腦，並開啟印表機電源。

3. 在電腦的光碟機中插入隨附的安裝光碟：

• Windows 10系統：電腦桌面右下角彈出“DVD RW 驅動器”視窗，滑鼠點擊此彈窗任意位置，然後點擊“Autorun.exe”，運行安裝程式。

8. 選擇“有線網路連接”，點擊“安裝”。

9. 安裝軟體自動搜索印表機，搜索過程可能需要一定時間。

10. 搜索完成後，選擇需要連接的印表機，點擊“下一步”。

注：

• 若已搜索到的印表機清單中，沒有您需要連接的印表機，請點擊“刷新”重新搜索。

• 主機名稱安裝印表機驅動在印表機IP位址發生變化時，您還可以直接列印，不需要重新安裝驅動。若需要以主機名稱方式安裝印表機驅動，請勾選 “以主機名稱安裝列表中的印表機”。

8. 安裝軟體自動安裝驅動，安裝過程可能需要一定時間，時間長短與您電腦配置有關，請您耐心等待。

9. 安裝完成，點擊“列印測試頁”，如果您的印表機能列印測試頁，說明您已經正確安裝印表機驅動。

3.3.Mac系統新增印表機

3.3.1.USB連接方式新增印表機

1. 使用USB連接線連接印表機和電腦，開啟電源。

2. 進入電腦的“系統偏好設定”－“印表機與掃描器”。

3. 點擊按鈕，選擇“加入印表機或掃描器”。

5. 點擊“加入”。

3.3.2. 有線網路連接方式新增印表機

1. 開啟印表機和電腦的電源。

2. 將電腦連接到有線網路。

3. 將網線連接到印表機網路介面，確保印表機連接到電腦所在的網路（如何配置網路，請參閱第4章）。

4. 進入電腦的“系統偏好設定”－“印表機與掃描器”。

5. 點擊按鈕，選擇“加入印表機或掃描器”。

6. 選擇印表機，然後從“使用”彈出服務表中選擇對應的印表機型號。

7. 點擊“加入”。

3.3.3. 無線網路連接方式新增印表機

1. 開啟印表機和電腦的電源。

2. 確保電腦與印表機已建立無線網路連接。

注：

• 若為基礎結構模式，將無線網路印表機連接到接入點（無線路由器），如何連接，請參閱第5.1章。

• 若為Wi-Fi熱點模式，將無線用戶端與無線網路印表機直接連接，如何連接，請參閱第5.2章。

3. 進入電腦的“系統偏好設定”－“印表機與掃描器”。

4. 點擊按鈕，選擇“加入印表機或掃描器”。

5. 選擇印表機，然後從“使用”彈出服務表中選擇對應的印表機型號。

6. 點擊“加入”。
